# FLOOD-DATASET-ANALYSIS
This project analyzes flood data to identify trends, causes, and impacts using Excel and Power BI.  

## Tools Used  
- Microsoft Excel  

## Objectives  
- Analyze seasonal flood patterns  
- Identify most affected regions  
- Provide data-driven insights for prevention strategies
